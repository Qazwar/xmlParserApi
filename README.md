# xmlParserApi
## Steps to Use API
## ------------------------
## Follow the below steps:
##-------------------------
### git clone https://github.com/ankithbti/xmlParserApi
### cd xmlParserApi
### mkdir -m 755 lib bin
### make
###
#################################
## You are ready to use the xmlParserApi
#################################
